# adarshmohanks.com

Personal portfolio for Adarsh Mohan, Product Designer.

The site is a single, dependency-free static page with no framework or build step.

## Files

- `index.html` — page content, styles, and interactions
- `Images/Adarsh.png` — profile image
- `Images/og-image.png` — 2400×1260 social sharing image
- `favicon.ico` — browser icon
- `apple-touch-icon.png` — iOS home-screen icon
- `robots.txt` and `sitemap.xml` — search-engine metadata
